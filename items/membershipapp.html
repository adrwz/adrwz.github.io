Membership App

FIRST PLACE AT STATES FOR MEMBERSHIP APP.







Overview

I developed an application designed to create, store, edit, and export members in a Java program using the JavaFX framework that combined Java with XML code, making it easy to build desktop applications. The application won first place at California FBLA States because of a well-designed program and a solid, confident pitch.

Process
Development

I spent three months building the application, working many hours after school on it. I spent time on working on my pitch deck to pitch to judges as well. The The application can add, delete, and edit users. It can also open and export as its own custom file. The app can create a report that can be exported as a CSV file or printed.

The application is built using SceneBuilder (which automatically generated the XML code) and JavaFX, which combined the XML code with the Java logic. A blue color scheme was used, per the competition organization's logo.
